# 기획 문서 (Docs)

번호순으로 정리해 전체 기획 흐름을 한눈에 파악할 수 있도록 구성했습니다.

| 번호 | 폴더 (영문명) | 내용 |
| --- | --- | --- |
| 01 | Game_Overview | 장르, 컨셉, 세계관, 핵심 루프, 경제 시스템 |
| 02 | System_Design | 튜토리얼, 러시타임, 제작 사이클, 오븐, AI, 손님, 매장관리 등 |
| 03 | Content_Design | 레시피, 이벤트, 시즌 컨텐츠 (추후 작성) |
| 04 | Level_Design | 맵 구조, 난이도 곡선, 레벨업 수치표 |
| 05 | UIUX_Design | 화면 구성, HUD, 팝업 흐름 (추후 작성) |
| 06 | Art_Concept | 캐릭터/배경 비주얼 방향, Meshy AI 프롬프트, Blender 파이프라인 |
| 07 | Sound_Design | BGM, SFX 컨셉 (추후 작성) |

> 폴더명은 영어로 통일했습니다 (한글 폴더명은 압축/윈도우 환경에 따라 글자가 깨지는 경우가 있어서요).
> 문서 내용(한국어)은 각 폴더 안에 넣으실 .docx/.md 파일 안에서 자유롭게 작성하시면 됩니다.

## 규칙
- 문서 수정 시 파일명 자체는 유지하고, 문서 내부에 변경 이력을 남기거나 Git 커밋 메시지로 이력 추적
- 큰 구조 변경이 필요하면 `_v2`, `_v3` 접미사로 버전 분리 후 최종본만 남기고 정리
